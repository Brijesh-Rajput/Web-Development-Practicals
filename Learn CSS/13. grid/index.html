Display:grid;

  • Parent (grid container) (display:grid) :-
      Properties:-
    • gap:-
      • row-gap,column-gap
    • grid-template-columns:-
      • To make different-different columns.
      • grid-template-columns:200px 300px 100px; ==> mans we will have 3 columns where 1st column width is 200, then 300 then 100
      • we can define width of block by using grid-template-columns
    • grid-template-rows:-
      • we can define height of block by using grid-template-rows
    • justify-content:-
      • x-axis ke through ham position kar sakte hain
      • center,end,space-around,space-between,space-evenly
    • align-content:-
      • Different from align-items.
      • center,stretch(bydefault => equally height will divided),start,end,space-evenly,space-between,space-around
    • align-items
  • Children:-
      Properties:-
    • justify-self:-
      • center,stretch(bydefault),start
    • align-self
    • grid-column:-
    • grid-row
    • grid-area
