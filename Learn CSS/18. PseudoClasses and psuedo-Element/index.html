Learn application of first-child, last-child, nth-child and ::before,::after in more details. where and how can we use ??

  • Pseudo Classes:-
    • used to define special state of an element.
    • As we can have different states of an element, therefore we can style different-different states of an element.
    • E.g:-
      • :visited, :link
      • :hover, :active
      • :first-child
      • :last-child
      • :nth-child
    • E.g:- link have these all states :- visited,hover,active,link
    • input has focus,hover states
    • button also have hover states
    • Also called as pseudo selectors
  • Pseudo Elements:-
    • used to style specific parts of an element.
    • E.g:-
      • ::first-letter, ::first-line, ::before, ::after, ::selection(to change the selection)
