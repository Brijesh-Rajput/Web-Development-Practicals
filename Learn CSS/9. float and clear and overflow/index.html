watch again for float and clear and learn it.

  • Overflow:-
    • it controls What happens when content is too big to fit into an area.
    • Values are :- visible(default value), hidden, scroll, auto
    • hidden will hide all the content which are overflowing.
    • scroll will give u right & bottom scroll bar whether content is overflowing or not.
    • auto will give u scroll bar onlyt when content overflows.
  • For Placeholder-images :- website name ==> placeholder.com
    • 1. https://via.placeholder.com/1200
    • 2. https://via.placeholder.com/1200x1000
  • Float:- (already seen demo in image.html)
    • How element should float
    • Values:- left,right,none,inherit
    • float to the left if value is left.
    • inherit will inherit the value of float properties from parent
  • clear:-
    • This property specifies what elements can float beside the cleared element and on which side.
    • Values:- left,right,both,none,inherit
    • float,clear should be used when we have to manage flow of image with text Don't missuse to make laylout's, for that we have display & position property like grid,flex,...
