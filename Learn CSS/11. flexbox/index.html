flex:- (flexbox)

  • flex used to make layouts. (as we have done jugaad for layout like position,float,clear ==> This is not a correct way.)
  • for using flex we need container,parent and children
  • Two types of properties of flex:-
    • Flex-container:-
      • Properties:-
      • flex-direction
        • row(bydefault),column,row-reverse(flex-items fills from right),column-reverse(fills from bottom to up)
      • flex-wrap
        • wrap,nowrap,wrapreverse
        • Its a children property:- gap(or row-gap, column-gap) ==> gap between children.
      • flex-flow
        • Shorthand property for flex-direction and flex-wrap
        • flex-wrap: wrap row
      • justify-content
        • Justify the children along main-axis
        • Valuess are:- space-between,space-around,space-evenly,flex-satrt(bydefault),flex-end,center
        • This is different from margin. firstly space between children will reduce when screen size reduce.
      • align-items
        • Align items used to change the position of children vertically
        • VAlues are:- center,stretch(bydefault),flex-start,flex-end
        • If we want to center a div in another div then we will make parent div as flex and justify-content:center and align-items:center
      • align-content
        • Align-content is uesed to manage the gap between two row of children in flex when screen size reduce.
        • flex-satrt,space-between,space-evenly are values
    • Flex-children:- (children property)
      • order
        • Default value is 0
        • Those box whose order value is high will go at last.
      • flex-grow:-
        • It's a important property./li>
        • we are telling a particular child that u should grow differently compare to all child.
        • Default value is 0
      • flex-shrink:-
        • Bydefault 1
        • shrinking will be more fast as its value increases.
        • if u want to see it(shrink) then commenting flex-flow:wrap
      • flex-basis:-
        • flex basis defines initial width in the direction of main axis for a particular child.
        • flex-basis:25%
        • flex-basis sets width b'coz flex-direction value is row.
        • flex-basis sets hieght b'coz flex-direction value is column.
      • flex:-
        • flex is a shorthand property for flex-grow,flex-shrink,flex-basis
        • flex:0 0 200px; /* 1st value is for flex-grow, 2nd for flex-shrink, 3rd is for flex-basis */
      • align-self:-
        • if we want that any children should align by itself on cross-axis then we will use align-self
        • flex-start,flex-end,center,scretch
