Selectors in HTML

Element Selector

The element selector selects HTML elements based on the element name.

Id Selector

The id selector uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page, so the id selector is used to select one unique element! A class cannot start with a number.

Class Selector

The class Selector selects HTML elements with a specific class attribute. To select elements with a specific class, write a period(.) character, followed by the class name. An id cannot start with a number.
