animation using transition property which applies on transform property.

It allows us to change property values smoothly, over a given duration.

to create transition effect :-two things must

  • css property u want to add an effect to(after effect which css property will change)
  • duration of effect
  • Note:- if duration not specified then transition will have no effect(b'coz default value is 0 of transition)

Property:-

  • transition-delay
  • transition-duration
  • transition-property
  • transition-timing-function:-
      Values are :-
    • ease:- (default) ==> transition effect with slow start,then fast,then end slowly
    • ease-in:- transition effect with slow start
    • ease-out:- transition effect with slow end
    • ease-in-out:- transition effect with slow start and end.
    • liner:- transition effect with same speed from start to end
    • cubic-bezier(n,n,n,n):- u defines ur own values
